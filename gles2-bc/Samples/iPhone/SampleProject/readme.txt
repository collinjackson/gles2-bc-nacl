Using the sample application
----------------------------
1. Create a new OpenGL ES Xcode application called SampleProject. 
2. Overwrite the sources from the newly created project directory with the sources from this directory
3. Delete the EAGLView.m and SampleProjectAppDelegate.m files from the Classes directory
4. In Xcode, Add the EAGLView.mm and SampleProjectAppDelegate.mm files to the project
5. In Xcode, Add the Sources/OpenGLES directory from the repository the the project
6. In Xcode, add all the shader files from OpenGLES/OpenGLES20/shaders to the build resources
7. Build and go
